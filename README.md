# Sudoku Game in C# 

---

## 🇺🇸 English

### 📖 Description
This is a simple Sudoku game developed using C#.  
It is designed to improve logical thinking and programming skills through a console-based puzzle system.

### 🎯 Features
- Sudoku grid structure
- User input handling
- Basic validation logic
- Console-based gameplay
- Logical puzzle solving practice

### 🛠 Technologies
- C# Programming Language
- .NET Console Application
- Arrays
- Methods
- Basic algorithms

### 🎮 Purpose
This project was created for learning purposes to practice:
- Logical thinking
- Problem-solving skills
- C# programming fundamentals
- Working with arrays and methods
